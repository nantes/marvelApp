# ZARA WEB CHALLENGE

Webapp to view all marvel characters

## Installation

Use npm  to install the necessary dependencies.

```bash
npm install
```

## Scripts

Run the application in dev mode

```bash
npm run dev
```

Generate a production build

```bash
npm run build
```

Locally preview the production build

```bash
npm run preview
```
